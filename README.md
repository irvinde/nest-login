# Login Nest y Mysql

1. Clonar proyecto
2. npm install
3. Clonar el archivo .env.template y renombrarlo a .env
4. Configurar las variables de entorno.
5. levantar la BD con concomando:
```
docker compose up -d
```
6. levantar el proyecto
```
npm run start:dev
```
